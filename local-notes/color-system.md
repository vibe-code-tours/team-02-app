# SonicSavor Color System

## Core Palette

| Role | Hex | Name | Usage |
|---|---|---|---|
| Primary | `#E85D04` | Burnt Orange | CTAs, active states, music waveform accent |
| Secondary | `#FFB703` | Golden Yellow | Highlights, hover states, rating stars |
| Accent | `#9D4EDD` | Deep Purple | Music-related elements, playlist badges |
| Background | `#0F0E17` | Dark Aubergine | Main background, cards surface |
| Surface | `#1A1926` | Midnight Slate | Cards, inputs, modals |
| Surface Elevated | `#242334` | Lighter Slate | Hovered cards, dropdowns |
| Text Primary | `#F5F3F0` | Warm White | Headings, main text |
| Text Secondary | `#A7A4B8` | Muted Lavender | Descriptions, labels |
| Success | `#2EC4B6` | Teal | Confirmation, meal loaded |
| Error | `#E63946` | Soft Red | Error toasts, API failures |

---

## Why These Colors

| Psychological Intent | Color Choice |
|---|---|
| Appetite stimulation | Orange + Gold (proven to trigger hunger) |
| Calm & trust | Deep aubergine background (reduces eye strain) |
| Premium feel | Dark surfaces with warm white text (Michelin menu vibe) |
| Musical energy | Purple accent (associated with creativity + sound) |
| Low-light comfort | No pure white, no pure black (late-night ordering friendly) |
